# Café Fontenebleau ☕

Um projeto de site institucional para uma cafeteria fictícia, desenvolvido como parte de um curso de desenvolvimento web para praticar e consolidar fundamentos de **HTML5** e **CSS3**.

## 🚀 Sobre o Projeto
O Café Fontainebleau foi criado com o objetivo de aplicar conceitos de semântica, estilização moderna e design responsivo. O foco principal foi construir uma interface elegante e funcional utilizando apenas as tecnologias nativas da web, sem o uso de frameworks externos.
